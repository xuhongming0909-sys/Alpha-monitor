## 五、数据源分层

```
获取 current_price 优先级：
  1. IOPV_FUTURES替代 中有映射 → Yahoo 5m 期货价
  2. 美股持仓 → Yahoo 5m + includePrePost
  3. 港股持仓 → Yahoo 5m (.HK)
  4. A股持仓 → Yahoo 5m (.SS/.SZ) → 腾讯行情 fallback
  5. 日股持仓 → Yahoo 5m (.T)
  6. 以上均失败 → DB 收盘价 (etf_prices / stock_prices)
```
