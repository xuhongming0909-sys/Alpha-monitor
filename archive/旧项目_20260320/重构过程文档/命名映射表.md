# 命名映射表

本文档用于锁定“英文旧名 -> 中文新名”的迁移规则。后续目录、文件、脚本、文档重命名均以本表为准。

## 1. 顶层目录

| 旧名称 | 新名称 | 说明 |
| --- | --- | --- |
| `数据抓取/` | `数据抓取/` | 只负责抓取、原始清洗、标准化 |
| `策略中心/` | `策略中心/` | 只负责业务判断、排序、筛选、计算 |
| `页面展示/` | `页面展示/` | 只负责 API 整形、前端脚本、页面模板 |
| `消息推送/` | `消息推送/` | 只负责推送格式、推送渠道、推送调度 |
| `共享基础/` | `共享基础/` | 只负责配置、路径、时间、日志、总线、公共模型 |
| `命令工具/` | `命令工具/` | 只保留工具脚本与对外命令入口 |
| `运行数据/` | `运行数据/` | 保存数据库、状态、缓存、快照 |
| `logs/` | `运行日志/` | 保存运行日志 |
| `重构文档/` | `重构文档/` | 保存需求、规格、计划、清单 |

## 5. 生态强制保留英文名

以下文件因运行时或生态工具约束，继续保留原名：

- `package.json`
- `package-lock.json`
- `requirements.txt`
- `.gitignore`
- `.env`
- `.env.example`
- Python 包入口文件 `__init__.py`
