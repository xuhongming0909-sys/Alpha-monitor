# 快速开始

## 1. 安装依赖

```powershell
npm install
pip install -r requirements.txt
```

## 2. 启动项目

```powershell
npm run dev
```

启动后访问：

- `http://127.0.0.1:5000/`

## 3. 你最常改的地方

- 改长期参数：
  - `config.yaml`
- 改推送样式：
  - `消息推送/样式/`
- 改页面接口：
  - `页面展示/接口/`
- 改前端页面：
  - `页面展示/模板/页面模板.html`
  - `页面展示/看板/启动服务.js`
- 改抓取逻辑：
  - `数据抓取/<插件名>/`
- 改策略逻辑：
  - `策略中心/<插件名>/`

## 4. 基础自检

```powershell
node --check 启动服务.js
python 数据调度.py exchange-rate
python 数据调度.py ah
python 命令工具/检查插件边界.py
```
