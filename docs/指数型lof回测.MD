## 五、结论

本次共回测 **19** 只基金（A类15只 + B类4只）。

1. **5只优秀**（MAE≤0.1%）：ETF与基准高度吻合，可直接用于生产IOPV计算
2. **3只良好**（MAE 0.1~0.3%）：可用，建议监控bias漂移
3. **3只一般**（MAE 0.3~1.0%）：需结合其他因素修正
4. **4只较差**（MAE>1.0%）：原油/中概类，需特殊处理

### 关键发现

- **原油类**（160723/161129/501018）误差最大，根本原因是USO跟踪原油期货而非现货，期货展期（contango）导致持续损耗
- **黄金**（160719）MAE=0.364%尚可接受，跟踪伦敦金现货价格
- **中概互联网**（164906）MAE=1.235%，港股+美股跨市场时间差+汇率导致偏差
- **美国大盘/行业ETF**（SPY/QQQ/XLY/XOP/IEO等）表现优秀，MAE均在0.1%左右或以下
- ETF映射优化后，RYH（161126）、IEO（162719）、IYR（160140）均获得完全匹配
