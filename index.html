正在检查本地服务状态...

这个入口页支持两种打开方式：如果本地服务已经启动，就直接进入 http://127.0.0.1:5000/； 如果你只是双击打开 HTML 文件，也能先看到本地预览页。

如果服务没有启动，请先执行 npm run start 或 npm run start:stable。

打开服务版页面 打开本地预览页

使用说明

  • 正式访问地址：http://127.0.0.1:5000/
  • 本地模板文件：presentation/templates/dashboard_template.html
  • 如果模板能打开但没有数据，通常说明前端文件没问题，只是后端服务还没启动。
