# 任务规格：手机端密集行表

## 类型

实现 / 规格同步 / 部署

## 目标

把 React UI 从“大块卡片”调整为“密集行表”：所有设备继续统一手机端导航，但内容区恢复一标的一行的表格扫读体验。

## 范围

- 更新正式规格，明确移动统一 UI 的主展示形态为密集行表。
- 调整 React 公共展示组件与样式，让每个股票、转债、基金、监控项以单行横向字段流展示。
- 保留 7 个底部标签与现有真实接口数据源。
- 更新相关 UI 测试、索引和记忆。
- 构建、提交、推送并部署服务器。

## 非范围

- 不恢复桌面端 1920px 锁死大表格。
- 不改后端接口、计算口径、数据抓取规则。
- 不恢复分红提醒、事件套利、推送设置到 React 导航。

## 正式规格影响

- `specs/spec.md`
- `specs/react-terminal-ui.md`
- 如自定义监控展示规则受影响，同步 `specs/custom-monitor.md`
