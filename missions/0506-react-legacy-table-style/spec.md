# 任务规格：React 还原旧网页端表格风格

## 类型

实现 / 规格同步 / 部署

## 目标

React 保留当前数据接线与底部导航，但内容区尽量还原旧网页端的干净清爽表格风格，取消复杂卡片感。

## 范围

- 更新正式规格，说明 React 内容区回到旧网页端风格的模块表格布局。
- 将各模块主展示切换为简洁表格。
- 保留 7 个底部标签和现有真实接口。
- 重新构建、测试、推送、部署。

## 非范围

- 不恢复旧 HTML 页面作为主入口。
- 不改后端接口和业务计算口径。
- 不恢复 React 顶部导航。

## 正式规格影响

- `specs/spec.md`
- `specs/react-terminal-ui.md`
