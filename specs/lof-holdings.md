# QDII LOF 基金标的清单

## 1. 总览

共 27 只 QDII LOF 基金，分三类估值方式：
- **A类-指数跟踪（24只）**：美股/商品/黄金/原油ETF直接映射
- **B类-T10持仓加权（3只）**：前十大持仓股票加权估值

注意：501312海外科技实际使用B类T10持仓法估值。

## 3. 估值公式

### A类
```
IOPV = NAV_T-2 x (1 + ETF_ret) x (FX_today / FX_T-2)
```
- ETF_ret = 标的ETF日收益率
- FX = 对应币种汇率

### B类
```
IOPV = NAV_T-2 x (1 + stock_ratio x sum(w_i x ret_i x fx_i))
```
- w_i = 第i只持仓权重（来自东财F10）
- ret_i = 第i只持仓日收益率
- stock_ratio = 股票仓位比例（来自东财pingzhongdata）
